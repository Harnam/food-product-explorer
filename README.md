# NutriLens

Try it out at https://nutrilens.harnam.in

A responsive food product explorer built using the OpenFoodFacts API.
Users can search for products, view nutritional information, and manage a simple cart.

This project demonstrates building a modern frontend application with Next.js, React Query, and Tailwind CSS, focusing on responsive UI, API integration, and state management.

## Features

- Product search by name or barcode
- Infinite scrolling product list
- Product detail page with nutritional information
- Nutrition grade display
- Category filtering
- Sorting options
- Add to cart functionality
- Cart overlay with item management
- Responsive design for mobile and desktop
- Loading skeletons for improved UX

## Tech Stack
- Next.js (App Router)
- React
- TypeScript
- TanStack React Query
- Tailwind CSS
- OpenFoodFacts API

## Installation

Clone the repository:

`git clone https://github.com/yourusername/nutrilens.git`

Install dependencies:

`npm install`

Run the development server:

`npm run dev`

Open in browser:

http://localhost:300
